Система учёта дикой природы заповедника

Десктопное приложение для управления данными о животных, сотрудниках, кормлениях и наблюдениях за здоровьем в условиях заповедной зоны. Поддерживает генерацию статистических и детальных отчётов в форматах PDF и Excel с корректным отображением кириллицы.

## Особенности

- Графический интерфейс на `customtkinter`
- Асинхронная работа с MySQL через `asyncmy`
- Генерация PDF-отчётов с поддержкой кириллицы (`reportlab` + шрифт DejaVuSans)
- Экспорт полных данных в Excel с диаграммами и аналитикой (`xlsxwriter`)
- Интерактивный дашборд с графиками (`matplotlib`)
- Безопасная обработка данных и валидация
- Полная локализация на русский язык

## Требования

**Системные требования**

- ОС: Debian 10/11/12, Ubuntu 20.04+
- Python: 3.9 или выше
- MySQL: 5.7+ или совместимая (например, MariaDB)
- Графическая среда: X11 (GNOME, XFCE и т.п.) — обязательно для GUI

**Зависимости Python**

См. [`requirements.txt`](#requirements-txt)

## Установка

 1. Клонирование репозитория
```bash
git clone https://github.com/ваш_логин/myproject.git
cd myproject
```
2. Создание виртуального окружения
```bash
python3 -m venv venv
source venv/bin/activate  # Linux/macOS
```
3. Установка зависимостей
```bash
pip install --upgrade pip
pip install -r requirements.txt
```
4. Установка системных зависимостей (Debian/Ubuntu)
```bash
sudo apt update
sudo apt install -y \
    fonts-dejavu-core \
    default-libmysqlclient-dev \
    build-essential \
    python3-dev
```
5. Настройка переменных окружения
``` bash
cp .env.example .env
nano .env  # при необходимости отредактируйте
```
Конфигурация
```txt
DB_HOST=pma.panteleeva.info
DB_PORT=3306
DB_USER=phpmyadmin
DB_PASSWORD=0907
DB_NAME=phpmyadmin
```
## Использование
Запуск приложения
```bash
source venv/bin/activate
python src/main.py
```
После запуска откроется главное окно с тремя вкладками:
- Главная — приветствие и кнопки экспорта
- Дашборд — аналитика и графики
- Управление данными — CRUD-операции по всем сущностям

 **Экспорт отчётов**
 - PDF: Статистический отчёт — содержит графики и ключевые метрики
 - PDF: Детальный отчёт — полные таблицы по всем разделам
 - Excel: Полный отчёт — данные + диаграммы + аналитика

Файлы сохраняются в рабочей директории с именами:
- статистический отчет.pdf
- детальный отчет.pdf
- wildlife_reserve_report_YYYYMMDD_HHMMSS.xlsx
